## Quest Setup Instruction

1. Duplicate (via right click and duplicate) the quest AINC_SceneManager_StoryManagerQuest_TEMPLATE. This is probably the only case right click and duplicate can be used.
2. Fix the quest name this will show on the log/quest notifications
3. Add any stages you need but do not remove and of them
4. On the quest alias remove any markers that you don't need and aren't places in your cell is a alias can't be filled the quest will be blocked from starting
5. You will need a handler quest script to perform work for the stages see [DissectionCamps_Spacers01](../../../../Papyrus/QOG/AdventurersInc/Quests/DissectionCamps_Spacers01.psc)
