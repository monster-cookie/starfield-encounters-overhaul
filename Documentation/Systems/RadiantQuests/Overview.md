# Radiant Quests: Overview

Instead of developing a custom system like I did for the unreleased Adventurers Inc Creation with Quarter Onion Games. I'm going to attempt to use the stock Story Manager/Radiant Quest system in Starfield. Though I'm pretty sure I will run into the same problems I did with ADVINC. The stock manager script is too complicated and conflicts with it self which is why the story manager area triggers seem to fail. 

## Features

Story Manager and the Radiant Quest System supports anything you can do with quest reference alias in the POI/Location currently attached to, spawning and controlling the loot of a "boss" chest, Spawning NPCs and Bosses using the MCS spawner system.

### Enabling/Disabling/Spawning Cell Objects

Using reference aliases and marker you can completely redesign and rename a location/poi to match your story telling needs.

### Story Chest

You can provide an important chest in the cell for Scene Manager to manage and a loot list for it to update the chest's contents with.  

### Spawning NPCs (and bosses)

Scene Manager uses the SQ_Spawner so it can spawn more NPCs and Bosses.
